# Ming (明)

> 知常曰明，不知常，妄作凶 —— 《道德经》

An open-source, model-agnostic AI agent that augments human System 2 thinking.

## Quick Start

```bash
# Install
pip install -e .

# Set API key
export MING_LLM_API_KEY="your-deepseek-api-key"

# Chat
ming "你好"

# Interactive mode
ming
```
